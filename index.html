Airport Runway Scheduling

Design and Analysis of Algorithms – Greedy Algorithm

Problem Statement

Airports have limited runways and multiple flights request landing or takeoff during specific time intervals. If two flights overlap on the same runway, it leads to delays and safety issues. The objective is to schedule the maximum number of non-overlapping flights.

Need for Algorithmic Solution

  • Manual scheduling causes conflicts
  • Checking all combinations is inefficient
  • Algorithm ensures optimal and fast scheduling

Solution Approach

We use the Greedy Algorithm (Activity Selection / Interval Scheduling) which selects flights based on earliest finishing time to maximize runway utilization.

Algorithm: Airport Runway Scheduling (Greedy Interval Scheduling)

  1. Sort all flights in non-decreasing order of their ending times
  2. Select the first flight (the one with the earliest finishing time)
  3. Let lastEnd = ending time of the selected flight
  4. For each remaining flight (in sorted order):
    • If start[i] ≥ lastEnd:
      • Select the flight
      • Update lastEnd = end[i]
  5. Stop when all flights are processed
  6. Output the selected flights

Pseudocode

ActivitySelection(start[], end[], n)
{
    sort activities by end time

    select first activity
    lastEnd = end[0]

    for i = 1 to n-1
        if start[i] >= lastEnd
            select activity i
            lastEnd = end[i]
}

C++ Implementation (Greedy Algorithm)

#include <iostream>
#include <algorithm>
using namespace std;

struct Flight {
    string id;
    int start;
    int end;
};

bool compareFlights(Flight a, Flight b) {
    return a.end < b.end;
}

void runwayScheduling(Flight flights[], int n) {
    sort(flights, flights + n, compareFlights);

    int lastEnd = -1;
    int count = 0;

    cout << "Scheduled Flights:\n";

    for (int i = 0; i < n; i++) {
        if (flights[i].start >= lastEnd) {
            cout << flights[i].id << " (" 
                 << flights[i].start << " - " 
                 << flights[i].end << ")\n";
            lastEnd = flights[i].end;
            count++;
        }
    }

    cout << "Total flights scheduled: " << count;
}

int main() {
    int n;
    cin >> n;

    Flight flights[n];

    for (int i = 0; i < n; i++) {
        cin >> flights[i].id
            >> flights[i].start
            >> flights[i].end;
    }

    runwayScheduling(flights, n);
    return 0;
}
